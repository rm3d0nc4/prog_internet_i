Gradiente radial

Gradiente radial (radial-gradient()) é uma função CSS que cria uma imagem com uma transição progessiva, no formato de um círculo, entre cores a partir de um ponto central do elemento.

  • Gradiente 1: Duas cores, direção definida como o valor padrão (center)
  • Gradiente 2: Gradiente no forma de uma elipse com ponto central a 20% da altura e largura do elemento, com cada cor sendo aplicada a apritr de uma porcentagem do raio gradiente
  • Gradiente 3: Duas cores, definindo o ponto central do elemento a 30% da altura e largura
  • Gradiente 4: Duas funções que definem gradientes que se espalham do ponto central das extremidade verticais, até a borda
